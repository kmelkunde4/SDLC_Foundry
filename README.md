\# SDLC Foundry



SDLC Foundry is an AI-powered multi-agent platform that accelerates the RPA Software Development Life Cycle (SDLC) from requirements analysis to bot generation, testing, and deployment readiness.



\## Team

Foundry Engineers



\## Planned Agents



1\. Requirement Analyzer

2\. PDD/SDD Generator

3\. Architecture Planner

4\. Workflow Skeleton Builder

5\. Test Case Generator

6\. Test Execution Agent

7\. CI/CD Agent (Optional)



\## Current Use Case



Employee Salary Analysis and Reporting Process



Process Flow:



Employee Excel File → Data Validation → Salary Analysis → High Salary Report Generation → Email Notification
